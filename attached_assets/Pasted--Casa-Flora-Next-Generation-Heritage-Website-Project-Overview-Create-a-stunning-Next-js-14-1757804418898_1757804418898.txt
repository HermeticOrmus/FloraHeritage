# Casa Flora - Next Generation Heritage Website

## Project Overview
Create a stunning Next.js 14 website for Casa Flora, a 100+ year heritage property in Boquete, Panama. Focus on glassmorphism design with sophisticated animations that tell the story of four generations of family stewardship.

## Technical Stack
- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS + Custom CSS for glassmorphism
- **Animations**: Framer Motion + GSAP for advanced effects
- **Backend**: Supabase integration
- **Deployment**: Replit hosting

## Brand Colors (Pantone)
- **Primary Blue**: #6B9BC3 (Pantone 2148 C - mountain sky)
- **Sage Green**: #A8B5A0 (Pantone P 149-10 C - lush vegetation)
- **Supporting**: White, soft grays, warm earth tones

```tsx

<motion.div 
  className="glass fixed bottom-8 right-8 p-6"

>
  {/* Multi-step booking form */}

```

#### Heritage Timeline
```tsx
// Scroll-triggered timeline with GSAP
useEffect(() => {
  gsap.registerPlugin(ScrollTrigger);
  
  gsap.timeline({
    scrollTrigger: {
      trigger: ".timeline",
      start: "top center",
      end: "bottom center",
      scrub: 1
    }
  })
  .to(".timeline-item", {
    opacity: 1,
    y: 0,
    stagger: 0.3
  });
}, []);
```

## GSAP Setup for Replit
```javascript
// _app.tsx - Initialize GSAP
import { useEffect } from 'react';
import { gsap } from 'gsap';
import { ScrollTrigger } from 'gsap/ScrollTrigger';
import { TextPlugin } from 'gsap/TextPlugin';

function MyApp({ Component, pageProps }) {
  useEffect(() => {
    gsap.registerPlugin(ScrollTrigger, TextPlugin);
    
    // Set GSAP defaults for consistent feel
    gsap.defaults({
      duration: 1.2,
      ease: "power2.out"
    });
    
    // Refresh ScrollTrigger on route changes
    const handleRouteChange = () => {
      ScrollTrigger.refresh();
    };
    
    return () => {
      ScrollTrigger.killAll();
    };
  }, []);

  return <Component {...pageProps} />;
}
```
