## üìã Usage Guidelines

### For Web Implementation:
1. **Line Art Assets**: Import as SVG or high-res PNG with transparent backgrounds
2. **Color Application**: Apply Casa Flora accent colors via CSS, keeping illustrations monochrome
3. **Section Dividers**: Use 3:1 or 6:1 horizontal line art elements
4. **Corner Accents**: Deploy 1:1 ornamental outlines
5. **Overlays**: Layer transparent line art over glassmorphism backgrounds
6. **Backgrounds**: Use minimal geometric patterns with low opacity

### Design Principles:
- **White on transparent**: Elegant white line art on transparent backgrounds
- **Selective color**: Apply muted accent colors sparingly via CSS/filters
- **Elegant restraint**: Less is more - simple outlines over complex illustrations
- **Glassmorphism integration**: Line art complements, doesn't compete with glass effects
- **Heritage sophistication**: Classic botanical/architectural illustration style

### Technical Parameters:
- `--v 7` for latest Midjourney model
- `--style raw` for crisp, clean line work
- `--ar` ratios optimized for web responsive design
- `--tile` for seamless pattern generation
- Request "transparent background PNG" in prompts
- Specify "white lines on transparent background" for proper rendering

### Post-Processing:
1. Convert to SVG for scalability when possible
2. Apply Casa Flora color palette via CSS filters or stroke colors
3. Use low opacity (10-30%) for subtle background patterns
4. Layer multiple line art elements for depth
5. Combine with glassmorphism cards for sophisticated aesthetic
